# FaceTec Project

This project integrates the FaceTec Browser SDK for facial authentication and liveness detection.

## Requirements

- Node.js

## Steps to Run

1. Install Node.js  
   Download and install Node.js from https://nodejs.org/

2. Install project dependencies  
   ```bash
   npm install

3. Start the FaceTec Browser SDK
   ```bash
   npm run facetec-browser-sdk

# Notes

Run the commands from the project root directory.

Ensure any required FaceTec configuration or environment variables are set before running (add the approriate DeviceKeyIdentifier in Config.js).
